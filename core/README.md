# Core Module

This directory contains the foundational components of the MEGASUS Android Device Control Toolkit.

## Components

- **`engine.py`**: The core execution engine handling ADB interactions, device state, and underlying shell commands.
- **`auth.py`**: Role-based access control and session management for the toolkit.
- **`logger.py`**: Centralized logging system for capturing application events and errors.
- **`plugin.py`**: Logic for dynamically loading and managing external plugins.
