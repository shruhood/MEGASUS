# Modules

This directory contains all the operational modules that define the capabilities of the MEGASUS toolkit.

## Available Modules

- **`apps.py`**: Application management (install, uninstall, list).
- **`automation.py`**: Scripting and task automation capabilities.
- **`connection.py`**: Device connection management (TCP/IP, pairing, etc.).
- **`data.py`**: Extraction and management of device data (SMS, contacts, etc.).
- **`device.py`**: Device information gathering and control.
- **`files.py`**: File system operations (push, pull, explore).
- **`network.py`**: Network state manipulation and analysis.
- **`screen.py`**: Screen capture, recording, and simulated touch inputs.
- **`security.py`**: Security-related operations and exploit modules.
- **`surveillance.py`**: Camera, microphone, and location tracking capabilities.
