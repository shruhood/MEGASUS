## вҡЎ Quick Start

**Requirements:** Python 3.8+ В· ADB in PATH В· USB Debugging enabled on device

```bash
# 1. Clone
git clone https://github.com/shruhood/megasus-pro.git
cd megasus-pro

# 2. Install dependencies
pip install -r requirements.txt

# 3. Connect a device
adb devices                          # USB
adb tcpip 5555                       # OR go wireless:
adb connect 192.168.x.x:5555

# 4. Launch
python megasus.py
```

> **Android setup:** Settings вҶ’ Developer Options вҶ’ Enable USB Debugging
